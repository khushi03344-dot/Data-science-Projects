# Agriculture Crop Yield Analysis Dashboard

## Project Title

## Prepared By

## Project Overview

This project focuses on analyzing agriculture crop yield data using data cleaning techniques and interactive dashboard visualization. The objective is to identify crop production patterns, compare crop performance, and understand the impact of environmental and agricultural factors on crop yield.

The project aims to support farmers, researchers, and policymakers in making informed decisions to improve agricultural productivity.

## Objectives

* Analyze the factors affecting crop yield.
* Identify high-yield and low-yield crops.
* Study the impact of rainfall, fertilizer, and pesticide usage on production.
* Compare crop production across states and seasons.
* Build an interactive dashboard for better decision-making.

## Dataset Information

* **Total Records:** 19,689
* **Total Columns:** 10

### Dataset Attributes

| Attribute       | Description                          |
| --------------- | ------------------------------------ |
| Crop            | Name of the crop                     |
| Crop Year       | Year of cultivation                  |
| Season          | Growing season                       |
| State           | State where the crop was grown       |
| Area            | Cultivated area                      |
| Production      | Total crop production                |
| Annual Rainfall | Rainfall received during cultivation |
| Fertilizer      | Fertilizer usage                     |
| Pesticide       | Pesticide usage                      |
| Yield           | Crop yield produced                  |

## Data Cleaning Process

The following preprocessing steps were performed to improve data quality:

1. Removed the first 10 rows from the Pesticides column.
2. Removed the Pesticides column from the dataset.
3. Removed duplicate values from the Area column.
4. Converted numerical values into appropriate text format where required.
5. Replaced the State value "Goa" with "Punjab" for consistency in analysis.

## Key Insights

* Coconut, Sugarcane, and Banana recorded the highest average crop yields.
* Coconut, Rice, and Wheat contributed the most to total production.
* Kerala, Tamil Nadu, and Karnataka showed the highest production levels.
* Whole-year crops contributed more production compared to seasonal crops.
* Larger cultivated areas generally resulted in higher production.
* Rainfall and fertilizer usage significantly influenced crop yield.
* Agricultural productivity varied across different states and seasons.
* Dashboard visualization helped identify important trends and patterns.

## Conclusion
